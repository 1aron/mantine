## Checkbox, Radio and Switch improvements

[Checkbox](https://mantine.dev/core/checkbox/), [Radio](https://mantine.dev/core/radio/) and [Switch](https://mantine.dev/core/switch/)

## Inverted Slider

[Slider and RangeSlider](https://mantine.dev/core/slider/) components now support `inverted` prop:

<Demo data={SliderDemos.inverted} />

## Other changes

- [Table](https://mantine.dev/core/table/) component now supports `withBorder` and `withColumnBorders` props
- [NumberInput](https://mantine.dev/core/number-input/) component now supports `removeTrailingZeros` prop
